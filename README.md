# Student Grading System

A simple Python program that calculates a student's letter grade and academic status based on the entered score.

## Features

- Accepts the student's name
- Accepts a score between 0 and 100
- Validates user input
- Assigns a letter grade (A, B, C, D, F)
- Determines whether the student has passed or failed
- Displays a formatted report card

## Technologies

- Python 3

## Concepts Used

- Variables
- User Input
- if / elif / else
- Comparison Operators
- Logical Operators

## How to Run

1. Open the project in VS Code.
2. Run the following command:

```bash
python "sec 3 - scoring system.py"
```
